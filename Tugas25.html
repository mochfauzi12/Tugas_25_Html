Mengenal Semantic Elements di HTML5

  • Article

    • Ketika ingin membuat layout untuk block Artike-artikel yang ada di website kita, tag artikel bisa di gunakan dengan Elements header dan footer Karena tag artikel mempunyai tipe block, artinya Elements ini memiliki lebar sesuai lebar browser jika tidak menentukan lebarnya

  • Aside

    • Element ini bisa di gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel. Karena Aside memiliki type/sifat block maka lebarnya sesuai page jika tidak menentukan lebarnya.
